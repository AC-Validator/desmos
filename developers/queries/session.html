# Query a session

This query allows you to retrieve the details of a session having its id.

CLI

desmoscli query sessions session [id]

# Example
# desmosli query sessions session 66
1
2
3
4

REST

/sessions/{sessionId}

# Example
# curl http://lcd.morpheus.desmos.network:1317/sessions/66
1
2
3
4
